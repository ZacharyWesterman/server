All About Tag Queries

Tags

Anything in quotes is considered a tag.
Non-quoted words are also tags, but those must not have any spaces or odd characters in them.
For example:
  • "tag1 tag2" is a single tag, but tag1 tag2 are two separate tags.
  • "exclamation!" is a valid tag, but exclamation! will cause a syntax error.
Note that all tags must be separated by an operator. This is on purpose.
If you search for, say tag1 tag2, would you expect to get files with both tags or with either of them?
A better option is to just not guess, so you'd have to write either tag1 and tag2 or tag1 or tag2.

Important notes:

  • Tags will never begin or end with spaces, but they may contain spaces.
  • Tags and tag queries are case-insensitive, e.g. the tag TAG is the same as tag.
  • Files will never have duplicate tags.
